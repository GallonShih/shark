## Tag Definition

- 格式：`v{Large Version}.{Feature Version}.{Hotfix Version}` (Ex: v1.1.1)
- 定義：
    - Large Version: 若有**巨大**變動時，推進此版本號。
    - Feature Version: 新功能的版本號，如果有新功能上線，推進此版本號。
    - Hotfix Version: 若有緊急的 issue 需要處理，在修正後，推進此版本號。
